# Commercial fonts

This contents of this directory are intentionally excluded from the distributed template.
It may contain proprietary fonts required to reproduce the official look.

Required fonts (commercial):
- Gadugi for the main text font.
- Aptos for headings and specific elements.
- Calibri for selected UI-style text.
- Cambria Math (TTC) for math typesetting.

Expected files and paths:
- `Fonts/commercial/Gadugi/gadugi.ttf`
- `Fonts/commercial/Gadugi/gadugib.ttf`
- `Fonts/commercial/Aptos/aptos.ttf`
- `Fonts/commercial/Aptos/aptos-bold.ttf`
- `Fonts/commercial/Aptos/aptos-italic.ttf`
- `Fonts/commercial/Aptos/aptos-bold-italic.ttf`
- `Fonts/commercial/Calibri/calibri.ttf`
- `Fonts/commercial/Calibri/calibrib.ttf`
- `Fonts/commercial/Calibri/calibrii.ttf`
- `Fonts/commercial/Calibri/calibriz.ttf`
- `Fonts/commercial/Cambria/cambria.ttc`

If these files are not present, the template automatically falls back to:
- Noto Sans (Gadugi fallback).
- Roboto (Aptos fallback).
- Carlito (Calibri fallback).
- STIX Two Math (Cambria Math fallback).
